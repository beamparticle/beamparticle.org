Description

Defines the behaviour for beamparticle callback modules, to be used with the beamparticle_handler pattern. The behaviour is letting the caller specify how to handle common functions for beamparticle operations.

Specify the -behaviour(beamparticle_callback). module attribute and the rest can then be implemented.
