create/3

create(Id::beamparticle_callback:id() | undefined, V::beamparticle_fn_body(), State::state()) -> {false | true | {true, beamparticle_callback:id()}, state()}

Create a new entry. If the id is undefined, the user has not submitted an id under which to store the resource: the id needs to be generated by the model, and (if successful), returned via {true, GeneratedId}. Otherwise, a given id will be passed, and a simple true or false value may be returned to confirm the results.

The created resource is validated before this function is called.
