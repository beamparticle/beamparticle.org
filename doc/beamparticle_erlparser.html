evaluate_erlang_expression/1

evaluate_erlang_expression(ErlangExpression::string() | binary()) -> any()

Evaluate a given Erlang expression and give back result.

Only local functions are intercepted, while the external functions are pass through. This is dangerous, but necessary for maximum flexibity to call any function. If required this can be modified to intercept external module functions as well to jail them within a limited set.
