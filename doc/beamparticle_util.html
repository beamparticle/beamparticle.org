hex_binary_to_bin/1

hex_binary_to_bin(H::binary()) -> binary() | undefined | error

convert a hex (in binary format) to binary while assuming that the size of the hex binary must be a multiple of 2, where each byte is represented by two characters. This function uses fun hex_binary_to_bin_internal/2 internally, which throws an exception when the input hex is not a valid but this function catches exception and gives back error.
