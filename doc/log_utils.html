req_log/4

req_log(Status, Headers, IoData, Req) -> any()

Request logs provided as a cowboy hook.

Folsom metrics can be triggered by having the cowboy handler setting a given metrics_source in the meta values of the cowboy_req object. Counts for status codes returned will be accumulated. If adding a meta value named start_time, whose value is obtained by calling erlang:montonic_time(milli_seconds), time difference calculation will be done between this time value and the end of the response.

By setting log_enabled to true in the cowboy_req meta object, log lines for an HTTP status will be generated for each response.
